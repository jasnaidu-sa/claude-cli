FEATURE SPECIFICATION: COUNTER COMPONENT
========================================

OVERVIEW
--------

A simple, self-contained React counter component that displays a numeric value and provides buttons to increment, decrement, and reset the count. The component uses TypeScript, React hooks, and inline styles with no external dependencies.

REQUIREMENTS
------------

Functional Requirements:
- Display a counter value prominently in the center
- Three buttons below the counter: − (decrement), + (increment), Reset
- Increment button adds 1 to the current value
- Decrement button subtracts 1 from the current value
- Reset button sets the value back to 0
- Counter can go negative (no floor limit)
- Always starts at 0 (no configurable initial value)

Technical Constraints:
- TypeScript with React 18
- Functional component using useState hook
- No external dependencies
- Inline styles only (no CSS files/modules)
- No props required
- Named exports (not default exports)
