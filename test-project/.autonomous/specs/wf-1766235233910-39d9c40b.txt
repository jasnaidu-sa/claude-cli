FEATURE SPECIFICATION: COUNTER COMPONENT
========================================

OVERVIEW
--------

A minimal React counter component with increment, decrement, and reset functionality. The component is self-contained with no props, uses TypeScript and React hooks, and includes inline styling for simplicity.

REQUIREMENTS
------------

Functional Requirements:
- Display a counter value prominently in the center
- Three buttons below the counter: − (decrement), + (increment), Reset
- Increment button adds 1 to the counter
- Decrement button subtracts 1 from the counter
- Reset button sets the counter back to 0
- Counter can go negative (no minimum value constraint)
- Counter always starts at 0

Technical Constraints:
- TypeScript with React functional component
- Use useState hook for state management
- No external dependencies
- No props required
- Inline styles only (no CSS modules or external stylesheets)
- Named export from barrel file

ARCHITECTURE
------------

File Structure:
src/
├── Counter.tsx        # Main component

└── index.ts           # Export barrel

Component Design:
- Single functional component using useState for the count value
- Self-contained with no external dependencies
- Inline styles for layout (counter centered, buttons below)

--------------------
