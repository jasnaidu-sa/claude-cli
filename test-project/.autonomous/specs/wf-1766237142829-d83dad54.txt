FEATURE SPECIFICATION: COUNTER COMPONENT
========================================

OVERVIEW
--------

A minimal, self-contained React counter component with increment, decrement, and reset functionality. The component uses TypeScript, React hooks (useState), and inline styles with no external dependencies.

REQUIREMENTS
------------

Functional Requirements:
- Display a counter value prominently in the center
- Three buttons below the counter: - (decrement), + (increment), Reset
- Increment button adds 1 to the counter
- Decrement button subtracts 1 from the counter
- Reset button sets the counter back to 0
- Counter can go negative (no minimum value constraint)
- Always starts at 0 (no configurable initial value)

Technical Constraints:
- TypeScript with React functional components
- Use useState hook for state management
- Inline styles only (no CSS modules or external styling)
- No external dependencies
- No props required
- Jest and React Testing Library already configured in project
