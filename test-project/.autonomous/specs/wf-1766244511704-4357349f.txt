FEATURE SPECIFICATION: COUNTER COMPONENT
========================================

OVERVIEW
--------

A simple, self-contained React counter component with increment, decrement, and reset functionality. The component uses TypeScript and React hooks with inline styling, requiring no external dependencies.

REQUIREMENTS
------------

Functional Requirements:
- Display a numeric counter value prominently in the center
- Three buttons below the counter: - (decrement), + (increment), Reset
- Increment button adds 1 to the current value
- Decrement button subtracts 1 from the current value
- Reset button sets the value back to 0
- Counter can go negative (no minimum value constraint)
- Counter always starts at 0

Technical Constraints:
- TypeScript with React functional component
- Use useState hook for state management
- No props required (self-contained component)
- No external dependencies
- Inline styles only (no CSS files or CSS modules)
- Jest and React Testing Library already configured

Component Design:

- Export: Named export Counter from barrel file
