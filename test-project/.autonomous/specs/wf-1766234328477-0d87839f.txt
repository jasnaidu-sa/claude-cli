FEATURE SPECIFICATION: COUNTER COMPONENT
========================================

OVERVIEW
--------

A minimal, self-contained React counter component with increment, decrement, and reset functionality. The component uses TypeScript and React hooks with inline styling, requiring no external dependencies.

REQUIREMENTS
------------

Functional Requirements:
- Display a counter value prominently in the center
- Three buttons below the counter: − (decrement), + (increment), Reset
- Increment adds 1 to the current value
- Decrement subtracts 1 from the current value
- Reset sets the value back to 0
- Counter can go negative (no minimum value constraint)
- Always starts at 0 (no configurable initial value)

Technical Constraints:
- TypeScript with React functional component
- Use useState hook for state management
- Inline styles only (no CSS files or modules)
- No external dependencies
- No props required
- Named export pattern

ARCHITECTURE
------------

File Structure:
src/
├── Counter.tsx        # Main component implementation
├── Counter.test.tsx   # Unit tests with React Testing Library
└── index.ts           # Export barrel file

Component Design:
- State: Single useState<number> initialized to 0
- Handlers: Three functions - increment, decrement, reset
- Layout: Vertical stack with count display on top, button row below
- Styling: Inline styles for prominent count display and button layout
