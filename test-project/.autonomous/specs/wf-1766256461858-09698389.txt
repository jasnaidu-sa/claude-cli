FEATURE SPECIFICATION: COUNTER COMPONENT
========================================

OVERVIEW
--------
A simple, self-contained counter component built with React and TypeScript. The component displays a numeric value that can be incremented, decremented, or reset to zero. The counter has no upper or lower bounds and can display negative values.

REQUIREMENTS
------------

Functional Requirements:
- Display a counter value starting at 0
- Provide three interactive buttons:
  - Increment (+): Adds 1 to the current value
  - Decrement (−): Subtracts 1 from the current value
  - Reset: Returns the counter to 0
- Counter value can be negative (no lower bound)
- Counter value can be any positive integer (no upper bound)

Technical Constraints:
- Framework: React with TypeScript
- State Management: React hooks (useState only)
- Styling: Inline styles (no external CSS dependencies)
- Dependencies: No external libraries beyond React core
- Props: Component accepts no props
- Initial State: Always starts at 0
- Export Style: Named export only

UI Requirements:
- Counter value displayed prominently in the center
- Three buttons arranged horizontally below the counter value
- Button labels: − (decrement), + (increment), Reset
- Clean, functional layout with basic inline styling

File Structure:

Component Design:
- Type: Functional component
- State: Single number state for counter value
- Handlers: Three event handlers for increment, decrement, and reset operations
