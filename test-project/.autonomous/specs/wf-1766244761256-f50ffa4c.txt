FEATURE SPECIFICATION: COUNTER COMPONENT
========================================

OVERVIEW
--------

Build a minimal React counter component with TypeScript that displays a numeric value and provides increment, decrement, and reset functionality. The component uses inline styles and requires no external dependencies.

REQUIREMENTS
------------

Functional Requirements:
- Display a counter value prominently in the center
- Three buttons below the counter: - (decrement), + (increment), Reset
- Increment adds 1 to the current value
- Decrement subtracts 1 from the current value
- Reset sets the value back to 0
- Counter can go negative (no minimum value constraint)
- Always starts at 0 (no initial value prop)

Technical Constraints:
- TypeScript with React hooks (useState)
- No external dependencies
- Inline styles only (no CSS modules or external stylesheets)
- No props required - fully self-contained component
- Jest and React Testing Library already configured

ARCHITECTURE
------------

File Structure:
src/
├── Counter.tsx        # Main component
├── Counter.test.tsx   # Unit tests
└── index.ts           # Named export barrel

Component Design:
- Single functional component using useState hook
- State: count: number initialized to 0
- Three handler functions: increment, decrement, reset
- Inline styles object for layout and basic styling
