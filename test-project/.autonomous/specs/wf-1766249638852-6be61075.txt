FEATURE SPECIFICATION: COUNTER COMPONENT
========================================

OVERVIEW
--------

A minimal React counter component built with TypeScript and React hooks. The component displays a numeric counter with increment, decrement, and reset functionality. The counter starts at 0 and can go negative.

REQUIREMENTS
------------

Functional Requirements:
- Display a counter value prominently in the center
- Increment button (+) adds 1 to the counter
- Decrement button (−) subtracts 1 from the counter
- Reset button returns the counter to 0
- Counter can go negative (no minimum value constraint)
- Counter always starts at 0

Technical Constraints:
- TypeScript with React hooks (useState)
- No external dependencies
- No props required - self-contained component
- Inline styles only (no CSS modules or external stylesheets)
- Jest and React Testing Library already configured in project
